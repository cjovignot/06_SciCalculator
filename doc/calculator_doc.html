    Exponential function¶ ↑

    Exponential function can be solved only out of multiple calculations

    def exp():

    # ...
global calc_input
if "^" in calculation:
    print("exp:", calculation)

    regex = r"(\d+)\^(\d+)"
    exp = re.match(regex, calculation)
    if regex:
        base = int(exp.group(1))
        exponent = int(exp.group(2))
        exp_result = base ** exponent
        print(calculation,":", exp_result)
        result_text.set(exp_result)
        calc_input = str(exp_result)

    Solving calculation function¶ ↑

    Solving function can solve basics operations in one string.

    Authorized operations :

    • “*” : Multiplication

    • “/” : Division

    • “+” : Addition

    • “-” : Substraction

    def equal(calculation):

    global calc_input
global calc_input_text
inner = []

    *In case of the string starts with “-”* The program will check the first caracter and if it starts with “-”, it will add “0” before it

    if calculation[0] == '-':
    calculation = '0' + calculation

    *Replacing every signs conflicts from input* To solve input problems, program will replace caracters in some cases:

    • '+-' or '-+'

      # Replace '+-' or '-+' with '-'
pattern1 = r"[+-]-|[+-]-"
calculation = re.sub(pattern1, "-", calculation)

    • '–'

      # Replace '--' with '-'
pattern2 = r"--"
calculation = re.sub(pattern2, "-", calculation)

    • '++'

      # Replace '++' with '+'
pattern3 = r"\+\+"
calculation = re.sub(pattern3, "+", calculation)

    • '*-'

      # "*-" case
if "*-" in calculation:
    calculation = '-{}*{}'.format(calculation[3:], calculation[0])

print("signs solved", calculation)

    Solving scopes priorities¶ ↑

    In case of scopes appears in the input, program will extract the inner scopes content to send it to calculation

    If the program doesn't find any scopes, it will check if the string contains *- to solve negative issues for multiplication

    # "*-" case

        calculation = '-{}*{}'.format(calc_input[3:], calc_input[0])

        elif array[i] == "/":

            array.pop(i-1)
            array.pop(i)
            calc_input_text.set(array)
            print("///", array)

    Solving additions and substractions in other cases
